# 实验3 说明文档

本目录包含实验3的源码文件，主要实现了基于哈希表和Trie（字典树）的集合(Set)数据结构。此实验用于比较哈希表与Trie两种结构在大规模字符串（如域名）存储和查找上的性能与实现差异。

## 文件说明

- `DST_Set.c`  
  主程序入口，演示如何用哈希表和Trie分别插入、查找、删除域名字符串，并测试两者性能和正确性。

- `hash.h` / `hash.c`  
  定义和实现哈希表所需的基本结构与哈希函数。

- `hash_adt.h` / `hash_adt.c`  
  基于哈希表的集合ADT（抽象数据类型）相关操作，包括初始化、插入、查找、删除、销毁等。

- `trie.h` / `trie.c`  
  定义Trie树节点结构，及基础创建函数。
